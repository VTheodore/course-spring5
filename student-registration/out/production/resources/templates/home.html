My Blog - Spring 5 Demo

Spring 5 Web MVC Application with Bootstrap Styling

Learn More @ Github Wiki

My Blog Demo shows how to build a dynamic web application using Spring Web MVC Controllers, Views, Forms, error handling, I18N & L10N, themes etc.

Styling is done using Bootstrap responsive design framework.

Application provides following functionality:

  • Users can browse articles and view detailed information about each article.
  • Users can edit article information and add new articles to the blog.
  • Administrators can manage (create, edit user data and delete ) all registered users, as well as articles.
